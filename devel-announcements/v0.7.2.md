Subject: [PATCH] ratatui_ruby v0.7.2 released

ratatui_ruby v0.7.2 is out.

This patch release fixes styled text rendering and adds flexibility to
Tabs padding.

## Fixed

Styled text objects were rendering as Ruby inspect strings. If you
passed a Text::Span or Text::Line to certain widget parameters, you saw
something like `#<data RatatuiRuby::Text::Span...>` instead of styled
content. Now fixed for:

- Tabs divider
- Table highlight_symbol
- BarChart BarGroup label
- Chart Axis title
- Chart Axis labels
- Chart Dataset name

## Added

Tabs padding_left and padding_right now accept String or Text::Line in
addition to Integer. Pass a Line object for colored or decorated
padding. Integer still works as before (generates that many spaces).

No breaking changes.
