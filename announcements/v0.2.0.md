Subject: [ANN] ratatui_ruby v0.2.0 released

I am happy to announce the release of ratatui_ruby v0.2.0.

This gem brings the Rust Ratatui library to Ruby, allowing you to
build rich terminal interfaces. This version introduces new widgets,
better styling, and a simpler way to build applications.

What is new:

* Simpler Apps
  We added `RatatuiRuby.main_loop` to manage the event loop and
  application lifecycle. This makes it easier to write simple
  applications without boilerplate.

* Calendars
  The new `Calendar` widget displays monthly views for date-based data.

* Charts
  A flexible `Chart` widget replaces the deprecated `LineChart` and
  `BarChart`, giving you more control over your data visualization.

* List Styles
  You can now customize how selected items look in a `List` using
  `highlight_style` and `highlight_symbol`.

* Ruby Support
  The gem now works with every non-EOL version of Ruby.
